# sudden-death

```
＿人人人人人人＿
＞　突然の死　＜
￣ＹＹＹＹＹＹ￣
```

Generates "Sudden death".

## Getting Started
Install the module with: `npm install sudden-death`

## Examples

```javascript
var SuddenDeath = require('sudden-death');
var totsuzenNoShi = new SuddenDeath('Sudden death');
totsuzenNoShi.say(); // "Sudden death"
```
